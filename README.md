## What are Claude Skills?

These projects are **structured AI prompt systems** that run inside [Claude.ai](https://claude.ai). No coding required.

1. Clone the repo
2. Upload the `SKILL.md` file to a Claude Project as project knowledge
3. Start a conversation — the skill guides you

## Quick Start

```bash
# Example: Set up the Missouri Expungement skill
git clone https://github.com/dougdevitre/expunge-skill.git
# Open Claude.ai → Create Project → Upload SKILL.md → Start chatting
```
