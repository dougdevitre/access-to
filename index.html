What are Claude Skills?

These projects are structured AI prompt systems that run inside Claude.ai — no coding required.

1

Clone the repo

Download the skill from GitHub. The key file is SKILL.md — a structured prompt that teaches Claude a specialized workflow.

git clone https://github.com/dougdevitre/<repo-name>
2

Upload to Claude

Create a new Project in Claude.ai and add the SKILL.md file as project knowledge.

3

Start a conversation

The skill guides you through its capabilities. Ask it for help — it handles the rest.

Stay in the loop

Get notified when new tools launch, existing projects hit milestones, or speaking events are announced.

Why "Access"?

Every day, people fall through gaps — in courts, classrooms, housing markets, and service systems — not because solutions don't exist, but because the tools to navigate them are scattered, expensive, or built for insiders.

These projects exist to close those gaps with practical, AI-powered, open-source tools that meet people where they are.
