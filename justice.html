Access to Justice

Court preparation, co-parenting documentation, expungement navigation, and protection order support — trauma-informed and child-centered.

The problem

Over 70% of civil legal matters in the U.S. involve at least one unrepresented party. In family court, parents navigating custody, protection orders, and documentation often lack the tools and structure to present their case effectively.

Expungement-eligible Missourians frequently don't know they qualify — or can't afford the process to clear records that limit their housing, employment, and educational access.

Our approach

Every tool in this pillar is built around three principles: child-centered framing, trauma-informed language, and court-ready documentation standards.

These are AI-powered Claude skills — structured prompt systems that guide practitioners, advocates, and self-represented parties through complex legal navigation without providing legal advice.

How to use these tools

These are Claude Skills — structured AI prompt systems that run inside Claude.ai. To get started:

  1. Clone the repository from GitHub
  2. Upload the SKILL.md file to a Claude Project as a project knowledge file
  3. Start a conversation — the skill will guide you through its capabilities

No coding required. These tools work entirely through conversation with Claude.
